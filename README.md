---

## 🚀 Features

- Text-to-Image generation using AI
- Fast and high-quality image output
- Simple and clean user interface
- Multiple style support (realistic, anime, digital art, etc.)
- Download generated images instantly
- Responsive design (mobile + desktop)

---

## 🧠 How It Works

1. Enter a text prompt (example: "futuristic city at night")
2. AI processes the prompt
3. Generates a unique image
4. User can preview and download result
